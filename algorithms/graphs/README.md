# Graphs

- Single Source Shortest Path (SSSP)
  - Start node s
  - Task: Calculate the shortest paths from s to all other nodes
- All Pairs Shortest Path (APSP)
  - Task: Calculate the shortest paths between all pairs of nodes

## Algorithms

- BreadthFirstSearch
- BellmanFord
- Floyd-Warshall
- DepthFirstSearch
- Kruskal
- Prim
